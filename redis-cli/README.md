### Redis basic operation
- 
 1. **set** 
 2. **get**
 3. **pip**

 If you want save data to redis,following cmd will be used:
 ```
 set key value
 ```
 **Example:**
 ```
set sample 0001
 ```


 If you want get data to redis,following cmd will be used:
```
get key
``
 **Example:**
```
get sample
